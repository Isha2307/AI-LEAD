📋 SYSTEM OVERVIEW
═════════════════════════════════════════════════════════════════════════════

The Lead Analysis & Scoring system provides a complete workflow for qualifying 
and prioritizing sales leads using Google Gemini API.

Components:
  1. LeadAnalysisAgent    → Extracts structured insights from raw lead data
  2. LeadScoringAgent     → Prioritizes leads based on analysis (NEW)
  3. FastAPI Endpoints    → HTTP interface for both agents
  4. Gemini Integration   → AI-powered analysis and scoring


🔄 WORKFLOW PIPELINE
═════════════════════════════════════════════════════════════════════════════

RAW LEAD INPUT
    ├─ name, email, company, industry
    ├─ employee_count, lead_message
    └─ Additional context/notes
         ↓
         [STEP 1: LEAD ANALYSIS]
         └─ POST /api/v1/leads/analyze
            • Validates input
            • Calls Gemini API
            • Analyzes requirements, budget, timeline, urgency
            • Returns: 8 structured fields including analysis
         ↓
    ANALYSIS RESULT
    ├─ summary
    ├─ requirement, budget, timeline
    ├─ urgency, company_size, industry
    └─ pain_points
         ↓
         [STEP 2: LEAD SCORING]
         └─ POST /api/v1/leads/score
            • Validates lead + analysis
            • Calls Gemini API
            • Scores 0-100
            • Assigns priority (Hot/Warm/Cold)
            • Generates reasoning
         ↓
    PRIORITIZED LEAD OUTPUT
    ├─ Score: 0-100
    ├─ Priority: Hot 🔥 | Warm 🔶 | Cold ❄️
    ├─ Confidence: 0-100%
    ├─ Reasoning: [Array of reasons]
    └─ Recommendation: Action item for sales team

EXAMPLE 1: Single Lead (Python)
───────────────────────────────────────────────────────────────────────────

from backend.agents import lead_analyzer_agent, lead_scoring_agent

# Step 1: Analyze lead
analysis = lead_analyzer_agent.analyze_lead(
    name="John Doe",
    email="john@company.com",
    company="Company Inc",
    industry="Technology",
    employee_count=250,
    lead_message="We need AI solutions for our business..."
)

print(f"Analysis: {analysis.summary}")
print(f"Budget: {analysis.budget}")
print(f"Timeline: {analysis.timeline}")

# Step 2: Score lead
scoring = lead_scoring_agent.score_lead(
    name="John Doe",
    email="john@company.com",
    company="Company Inc",
    industry="Technology",
    employee_count=250,
    lead_message="We need AI solutions for our business...",
    analysis=analysis
)

print(f"Score: {scoring.lead_score}/100")
print(f"Priority: {scoring.priority}")
print(f"Confidence: {scoring.confidence}%")
print(f"Reasons: {scoring.reasoning}")

# Step 3: Get recommendation
recommendation = lead_scoring_agent.get_action_recommendation(scoring)
print(f"Action: {recommendation}")

EXAMPLE 4: API Endpoint (Python requests)
───────────────────────────────────────────────────────────────────────────

import requests

# Step 1: Analyze
lead_data = {
    "name": "John Doe",
    "email": "john@company.com",
    "company": "Company Inc",
    "industry": "Technology",
    "employee_count": 250,
    "lead_message": "We need help with..."
}

response = requests.post(
    "http://localhost:8000/api/v1/leads/analyze",
    json=lead_data
)

analysis_result = response.json()

# Step 2: Score
scoring_request = {
    **lead_data,
    "analysis": analysis_result["analysis"]
}

response = requests.post(
    "http://localhost:8000/api/v1/leads/score",
    json=scoring_request
)

scoring_result = response.json()
print(f"Score: {scoring_result['scoring']['lead_score']}/100")
print(f"Priority: {scoring_result['scoring']['priority']}")
